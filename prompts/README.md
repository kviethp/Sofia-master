# Sofia Agent Prompts

This directory contains the canonical Sofia Master role prompts used by the
multi-agent control plane.

Files in this directory are part of the published repository and the release
bundle. The agent-system conformance checks validate these prompts for:

- inventory completeness
- portable paths
- CLI alignment
- role coverage
