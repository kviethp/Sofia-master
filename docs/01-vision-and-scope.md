# Vision and Scope

## Vision

Sofia Master should become an open-source, easily shareable, self-host friendly process OS for AI-assisted software delivery.

## Scope

### In scope
- skill-driven orchestration
- workflow engine
- policy engine
- project memory
- artifact collection
- audit and usage ledger
- OpenClaw runtime integration
- 9Router routing integration
- doctor / preflight / conformance
- golden path demo
- OSS packaging and contributor workflow

### Out of scope for initial implementation
- enterprise billing
- hosted SaaS control plane
- advanced multi-tenant isolation
- full enterprise SSO
- advanced compliance exports

## User groups
- solo builders
- small product teams
- internal engineering organizations
- OSS contributors

## Success criteria
- a new user can install and run the golden path on one machine
- an internal team can route tasks through OpenClaw and 9Router with policy control
- configuration drift can be detected before task execution
- project state survives worker failure
- open-source contributors can understand the repo within one session
