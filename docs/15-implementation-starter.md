# Implementation Starter

## Goal

Provide enough runnable structure so that AI implementation agents converge on the same system shape.

## Starter scope
- monorepo scaffold
- package boundaries
- basic TypeScript configs
- initial database schema
- API server stub
- worker stub
- router client stub
- OpenClaw adapter stub
- doctor and smoke script placeholders
