# Implementation starter

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../15-implementation-starter.md).

## Mục đích

Định nghĩa điểm bắt đầu triển khai code từ pack tài liệu.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
