# Phân rã module

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../03-module-breakdown.md).

## Mục đích

Liệt kê các module chính, ranh giới trách nhiệm và dependency.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
