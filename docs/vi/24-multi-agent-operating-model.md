# Mô hình vận hành multi-agent

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../24-multi-agent-operating-model.md).

## Mục đích

Mô tả cơ chế điều phối khi phát triển với nhiều agent song song.

## Điểm mới cần chú ý

- Bộ prompt hiện tại không còn chỉ có 3 file gốc.
- Đã có thêm prompt riêng cho:
  - Runtime Audit and Reconciliation
  - Core Platform
  - Integrations
  - UI and Operator Experience
  - OSS and Release
- Bản tiếng Anh hiện dùng thêm [../34-role-skill-matrix.md](../34-role-skill-matrix.md) để đồng bộ prompt inventory, role inventory và nhóm skill khuyến nghị.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
