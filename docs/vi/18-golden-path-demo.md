# Golden path demo

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../18-golden-path-demo.md).

## Mục đích

Mô tả luồng demo chuẩn cần pass để chứng minh hệ thống hoạt động.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
