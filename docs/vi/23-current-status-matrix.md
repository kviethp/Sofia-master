# Ma trận trạng thái hiện tại

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../23-current-status-matrix.md).

## Mục đích

Tóm tắt trạng thái implemented, scaffold, draft của từng khu vực repo.

## Điểm mới cần chú ý

- Bản tiếng Anh hiện đã thêm khu vực `Role and skill coordination docs`.
- `packages/skill-schema` và `packages/skill-compiler` hiện đã có code thật để validate và compile skill.
- Điều còn thiếu là runtime trust enforcement end-to-end, không phải compile layer cơ bản.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
