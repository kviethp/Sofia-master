# Ma trận tương thích

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../09-compatibility-matrix.md).

## Mục đích

Nêu phạm vi tương thích giữa runtime, provider, môi trường và deployment mode.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
