# Kế hoạch kiểm thử conformance

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../08-conformance-test-plan.md).

## Mục đích

Mô tả các nhóm kiểm thử dùng để chứng minh hành vi runtime và durability.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
