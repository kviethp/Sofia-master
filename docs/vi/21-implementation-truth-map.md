# Truth map triển khai

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../21-implementation-truth-map.md).

## Mục đích

Xác định nguồn chân lý cho từng phần để giảm drift khi nhiều agent cùng làm.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
