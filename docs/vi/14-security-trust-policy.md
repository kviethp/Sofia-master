# Security và trust policy

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../14-security-trust-policy.md).

## Mục đích

Tập hợp nguyên tắc trust boundary, redaction và bảo mật ở mức hệ thống.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
