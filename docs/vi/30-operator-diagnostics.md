# Operator diagnostics

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../30-operator-diagnostics.md).

## Mục đích

Hướng dẫn dùng diagnostics để kiểm tra trạng thái hệ thống khi vận hành.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
