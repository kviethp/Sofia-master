# Playbook incident response

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../27-incident-response-playbook.md).

## Mục đích

Đưa ra quy trình chẩn đoán và xử lý sự cố runtime.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
