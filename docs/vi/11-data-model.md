# Mô hình dữ liệu

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../11-data-model.md).

## Mục đích

Mô tả task, run, approval, artifact, decision, usage và các bảng liên quan.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
