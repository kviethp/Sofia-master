# Kiểm kê runtime hiện có

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../06-existing-runtime-audit.md).

## Mục đích

Hướng dẫn kiểm tra môi trường runtime hiện tại trước khi đưa Sofia vào vận hành.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
