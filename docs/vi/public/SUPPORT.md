# Hỗ trợ

Ngôn ngữ:
- [English](../../public/SUPPORT.md)
- [Tiếng Việt](./SUPPORT.md)

Bản tiếng Anh là nguồn chuẩn cho quy trình hỗ trợ.

## Sử dụng kênh nào

- Dùng Issues cho lỗi và yêu cầu tính năng
- Dùng Discussions cho câu hỏi thiết kế
- Dùng kênh bảo mật riêng cho báo cáo nhạy cảm

## Với operator self-host

Khi gửi yêu cầu hỗ trợ, nên kèm:

- phiên bản Sofia
- phiên bản Node
- output từ `node apps/sofia-api/scripts/doctor.js`
- output từ `node apps/sofia-api/scripts/compatibility-snapshot.js`
- task ID, run ID và approval ID liên quan nếu có
