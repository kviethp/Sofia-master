# Agent handoff

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../20-agent-handoff.md).

## Mục đích

Quy định cách bàn giao công việc giữa các agent và checkpoint review.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
