# API contracts

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../12-api-contracts.md).

## Mục đích

Tóm tắt contract API và cách các surface runtime tương tác với nhau.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
