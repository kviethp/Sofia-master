# Ownership map

Bản tiếng Việt cho tài liệu [gốc tiếng Anh](../22-agent-ownership-map.md).

## Mục đích

Phân quyền sở hữu module và write-scope cho nhiều agent.

## Cách dùng

- Đọc bản này để nắm nhanh ý chính bằng tiếng Việt.
- Dùng tài liệu tiếng Anh làm nguồn chuẩn khi cần chi tiết kỹ thuật hoặc khi có khác biệt về diễn đạt.
