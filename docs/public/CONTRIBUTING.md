# Contributing

## Development flow
1. fork the repo
2. run `node scripts/bootstrap.mjs`
3. run `pnpm install`
4. run doctor and smoke
5. work on a focused slice
6. add tests
7. update docs

## Quickstart

Use [28-quickstart.md](./docs/28-quickstart.md) as the default local setup path.

## Pull request expectations
- explain change
- note affected module
- include tests
- include compatibility impact
- include migration impact if any
