# Support

Use:
- Issues for bugs and feature requests
- Discussions for design questions
- Security contact for sensitive reports

## Self-host operators

When opening support requests, include:

- Sofia version
- Node version
- output from `node apps/sofia-api/scripts/doctor.js`
- output from `node apps/sofia-api/scripts/compatibility-snapshot.js`
- affected task IDs, run IDs, and approval IDs when relevant
